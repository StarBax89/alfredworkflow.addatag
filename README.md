alfredworkflow.addatag

1. Install [Tag](https://github.com/jdberry/tag/ "Tag")
2. Install workflow to  alfred
3. Tag ;)

Add a tag to files selected in finder.
- select files
- type "ftag" to alfred
- type the argument (argument is the name of the tag)
